# 🍳 さきちゃんのレシピ帳

Instagramで「作りたい！」と思った料理レシピを、URLや写真で保存しておける個人用アプリ。

**🌐 公開URL:** https://mitsukenn.github.io/recipe-stock/

**📖 つかいかたガイド:** [MANUAL.md](./MANUAL.md)

---

## ✨ 機能

- 📸 写真 / URL / メモ / タグで料理を保存
- 🏷️ カテゴリーを自由にカスタマイズ（追加・編集・削除）
- 🍳 「作ったよ！」スタンプで作った回数・日付を記録
- 📅 月別タブ・カテゴリーフィルタ・テキスト検索
- ✨ 30日以上前に保存したけど作ってないものを「こんなの保存してたよ」でランダムリマインド
- 💾 JSONでバックアップ・復元

## 🛠️ 技術構成

- React 18（CDN）+ Babel Standalone（ビルド不要の単一HTMLアプリ）
- データ保存：localStorage（クライアント完結、バックエンドなし）
- PWA対応：ホーム画面追加でアプリっぽく動作
- ホスティング：GitHub Pages

## 📂 ファイル構成

```
├── index.html      # アプリ本体（すべてここに入ってる）
├── icon.png        # アプリアイコン（1024×1024）
├── manifest.json   # PWAマニフェスト
├── MANUAL.md       # ユーザー向けマニュアル
└── README.md       # これ
```
